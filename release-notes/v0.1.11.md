# Claude Code A+BAY v0.1.11

This release adds the Codex-inspired Review sidebar for desktop Git workflows.

## Changes

- Added a right-side Review panel that shows local Git changes for the active workspace.
- Added file-level additions, deletions, status badges, and inline diff previews.
- Added Git actions for refreshing changes, staging all files, unstaging files, and reverting unstaged edits with confirmation.
- Added desktop Tauri commands for reading Git status and running review actions from the app.
- Kept the release as the standard SemVer successor to v0.1.10 for Tauri updater compatibility.
