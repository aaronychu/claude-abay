# Claude Code Haha v0.2.2

这个版本聚焦于桌面端视觉体验和主题系统的增强。

## Highlight

- macOS Liquid Glass 支持：sidebar 使用 macOS 26 专用材质，提供原生磨砂透明效果。
- 跟随系统配色：新增「跟随系统」主题选项，自动匹配系统亮/暗模式。
- 智能主题适配：当手动选择的配色与系统不匹配时，sidebar 自动叠加半透明 overlay 确保文字可读性。
- SVG Logo：应用名称改为 SVG 矢量图，自动随主题变色。

## 主要更新

- macOS sidebar 改用 `window-vibrancy` crate 的 `apply_liquid_glass` API，使用 `NSGlassEffectViewStyle::Sidebar` 专用材质。
- 新增「跟随系统」配色选项（默认），通过 `prefers-color-scheme` 媒体查询实时跟随系统亮/暗模式。
- 当手动选择的配色与系统不匹配时，sidebar 自动叠加半透明 overlay（暗色 0.60 / 亮色 0.55），确保文字在原生 vibrancy/mica 背景上清晰可读。
- Windows 配置新增 Mica 效果支持。
- 应用名称从纯文字改为 SVG 矢量图（`LCText.svg`），通过 CSS `filter: invert()` 随主题自动变色。
- 侧边栏右键菜单（重命名/删除）改为 Portal 渲染，修复被 sidebar 裁剪不可见的问题。
- 清理工作区和终端面板的多余 border，界面更简洁。

## 问题修复

- 修复 sidebar 右键菜单因 `overflow: hidden` 和 `will-change` 导致不可见的问题。
- 修复 macOS sidebar 透明效果需要 `macOSPrivateApi: true` 才能生效的问题。
- 修复 auto-generated stub 缺少 `handleHahaOAuthApi` 导致 build 失败的问题。
