# Claude Code A+BAY v0.1.10

This release polishes the desktop interface and fixes the Codex-inspired light and dark theme pass.

## Changes

- Refined the desktop light and dark themes across the sidebar, tab bar, modals, dropdowns, buttons, settings, empty state, and active session view.
- Fixed dark-mode segmented controls in Settings so selected Chinese labels remain readable.
- Restyled the chat composer toolbar so it no longer appears as a misplaced blue block in dark mode.
- Enabled macOS transparent window support and sidebar window effects for a more native translucent desktop sidebar.
- Updated local macOS packaging so Apple Silicon builds include the transparent window configuration.
